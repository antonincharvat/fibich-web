Music player for macOS

Music. Organized.

Fibich is a simple yet powerful macOS music player focused on organizing your library and returning to your favorite tracks without the interface getting in the way.

Current feature set

Made for serious local libraries.

Fibich combines wide format support, durable playlists, library organization, metadata tools, and macOS playback integration in one native player.

Native macOS player

  • Split layout for library browsing, playlists, artwork, details, and playback controls.
  • Built for local music libraries on macOS.

Broad audio support

  • Plays FLAC, WAV, AIFF, ALAC/M4A, MP3, Opus, Ogg, WavPack, APE, Musepack, TTA, Shorten, DSD, and more.
  • Supports tracker modules such as XM, IT, MOD, and S3M.
  • Shows CUE sheets as virtual tracks with segment playback and badges.

Library management

  • Add persistent library folders.
  • Automatically scan folders and keep the catalog in sync.
  • Search, sort, browse folder trees, and rescan on demand.
  • Send selected folders or tracks to the active/new playlist.

Playlist workflow

  • Use multiple playlist tabs with rename and drag-reorder support.
  • Restore playlists, tabs, columns, pane sizes, and scroll positions.
  • Drop external files or folders into playlists without adding library roots.
  • Customize columns and view track, folder, duration, and selection totals.

Playback controls

  • Play, pause, stop, skip, seek, mute, and adjust volume.
  • Stop after the current track.
  • Toggle total or remaining time in the scrubber.
  • Use macOS media keys and Now Playing controls.
  • Resume the last session and playback position on launch.
  • Control playback from the menu bar popover.

Metadata and artwork

  • View selectable track metadata and detailed audio properties.
  • Show embedded artwork or common cover images from album folders.
  • Edit tags, lyrics, ratings, identifiers, and custom fields.
  • Batch edit selected tracks while preserving untouched values.
  • Validate numeric fields and custom metadata keys.
